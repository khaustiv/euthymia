### 3. Запуск ###
1. Клонировать репозиторий:
```
git clone https://github.com/khaustova/euthymia.git
```
2. Запустить проект с помощью докера:
```
docker-compose up --build
```
Веб-приложение запустится на http://127.0.0.1:8000.
